Enquiry Logging Setup

Route every contact-form submission to your email and an Excel sheet — free, no server required

This one-time setup takes about 10 minutes. After it's done, every enquiry submitted on sindhucorporation.com will:

  • Email you (and optionally a second person) with the enquiry details
  • Add a new row to a Google Sheet (which you can download as .xlsx anytime)
  • Auto-assign a "New" status so you can track follow-ups
What you need: a Google account (any Gmail works), plus the two files already created for you in the admin/ folder of this project.
admin/enquiry-log.xlsx

The Excel template — Enquiries sheet, Dashboard, and instructions. Open this once to see the structure.

admin/google-apps-script.gs

The webhook code — receives form data, writes it to the sheet, emails you.

Step-by-step

1

Upload the Excel to Google Drive

  1. Go to drive.google.com.
  2. Click New → File upload and choose admin/enquiry-log.xlsx.
  3. When it appears in your Drive, right-click it → Open with → Google Sheets.
  4. Google Sheets will make a linked Google Sheet copy. Keep this tab open — we'll work in it.
2

Open the Apps Script editor

  1. Inside the Google Sheet, click Extensions → Apps Script.
  2. A new tab opens with a file called Code.gs. Delete everything inside it.
  3. Open admin/google-apps-script.gs on your computer, select all, copy, and paste it into the editor.
  4. Click the floppy-disk Save icon. Name the project "Sindhu Enquiry Webhook".
3

Put your email address in the script

Near the top of the pasted code you'll see: 
const NOTIFY_EMAIL   = 'info@sindhucorporation.com';
const NOTIFY_EMAIL_2 = '';
Change these to the email address(es) that should receive enquiries. Save again.
4

Deploy the script as a Web App

  1. In the Apps Script editor, click Deploy → New deployment.
  2. Click the gear icon → choose Web app.
  3. Fill in:
    • Description: "Sindhu Enquiry Webhook v1"
    • Execute as: Me (your Google account)
    • Who has access: Anyone
  4. Click Deploy. Google will ask for permissions — approve them (click "Advanced" if you see a warning, then "Go to Sindhu Enquiry Webhook (unsafe)" — it's your own script, it's safe).
  5. You'll see a Web app URL. It looks like: 
    https://script.google.com/macros/s/AKfyc............/exec
    Copy it.
Tip: Paste the URL into a browser tab. You should see {"ok":true,"service":"Sindhu Corporation Enquiry Webhook"}. That confirms the webhook is live.
5

Wire the website to the webhook

  1. Open assets/js/main.js in your editor.
  2. Near the top you'll find: 
    const ENQUIRY_WEBHOOK_URL = '';
  3. Paste your Web app URL between the quotes: 
    const ENQUIRY_WEBHOOK_URL = 'https://script.google.com/macros/s/AKfyc..../exec';
  4. Save the file and re-upload to your hosting (or commit and push if using GitHub Pages / Netlify / Vercel).
6

Test it end-to-end

  1. Open contact.html on your live site.
  2. Submit a test enquiry (use your own name/email so you can verify).
  3. Within a few seconds you should:
    • See an email in your inbox with subject "🏠 New Enquiry #1 — ..."
    • See a new row in the Google Sheet's Enquiries tab with status "New"
    • See the Dashboard tab counters tick up

Daily use

✓

Managing leads

Work directly in the Google Sheet. Edit the Status column as you progress each lead:
  • New → just arrived
  • Contacted → you've called/emailed them
  • Site Visit → visit booked
  • Booked → deal closed
  • Closed - Lost → did not convert
Colour-coding will update automatically.
⬇

Download as Excel anytime

In the Google Sheet: File → Download → Microsoft Excel (.xlsx). You now have a local .xlsx backup of every enquiry.

That's it.

From the moment the webhook URL is pasted into main.js and the site is re-deployed, every enquiry flows automatically into your inbox and the Excel log. No manual copying required.
